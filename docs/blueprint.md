# **App Name**: FinanceFlow

## Core Features:

- Manual Income Input: Allow users to manually input income data directly within the app.
- Manual Expense Input: Allow users to manually input expense data directly within the app.
- Dashboard Visualization: Display visual charts to represent income, expenses, and savings over time.
- Spending Categorization: Automatically categorize expenses based on keywords in the description.
- Budgeting Tool: Set monthly budget goals and compare actual spending against these goals using generative AI as a tool to assess when/if budget corrections are needed.
- Report Generation: Create a summary of the user's financial activity for a given period. Can be exported in PDF or CSV formats

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calm, organized feel.
- Background color: Pale cream (#F5F5DC), providing a warm and inviting canvas.
- Accent color: Muted rose (#D87093) for highlighting key elements.
- Body and headline font: 'Literata', a transitional serif with a literary feel, suitable for both headlines and body text.
- Use clear, minimalist icons for each expense category, improving visual understanding.
- Implement a clean, dashboard-style layout for displaying financial data, inspired by modern financial apps.
- Add subtle animations when the data changes.
