# XpenseLab Brand Kit

Complete brand assets and guidelines for XpenseLab.

## Logo Variations

### 1. Primary Logo (Stacked)
- **Use**: Main brand identity, dark backgrounds
- **Format**: SVG, PNG
- **Files**: `logo-primary-stacked.svg`, `logo-primary-stacked.png`

### 2. Primary Logo (Light Background)
- **Use**: Light backgrounds, print materials
- **Format**: SVG, PNG
- **Files**: `logo-primary-light.svg`, `logo-primary-light.png`

### 3. Monochrome Logo
- **Use**: Single-color applications, fax, embossing
- **Format**: SVG, PNG
- **Files**: `logo-monochrome-dark.svg`, `logo-monochrome-light.svg`

### 4. Horizontal Lockup
- **Use**: Navigation bars, headers, wide spaces
- **Format**: SVG, PNG
- **Files**: `logo-horizontal.svg`, `logo-horizontal.png`

### 5. App Icon / Favicon
- **Use**: App icons, favicons, social media avatars
- **Format**: SVG, PNG (square 1:1)
- **Files**: `icon-app.svg`, `icon-app.png`, `favicon.ico`

- **Usage**: Headings, body text, UI elements

- **Font**: Manrope Bold

## Logo Icon

The logo icon consists of two overlapping rounded bars:
- **Bar 1**: Rotated 35deg, full height, gradient from electric-blue to emerald-green
- **Bar 2**: Rotated -35deg, 80% height, gradient from electric-blue via primary to emerald-green

## Usage Guidelines

### Minimum Sizes
- **Primary Logo**: Minimum 32px height
- **App Icon**: Minimum 16x16px (favicon), 512x512px (app store)

### Clear Space
- Maintain clear space equal to the height of the icon around all sides of the logo

### Do's
✅ Use full-color logos on dark backgrounds
✅ Use light background versions on light surfaces
✅ Use monochrome versions for single-color applications
✅ Maintain aspect ratio when scaling
✅ Use horizontal lockup in navigation bars

### Don'ts
❌ Don't rotate or distort the logo
❌ Don't change colors (except monochrome versions)
❌ Don't place logo on busy backgrounds without sufficient contrast
❌ Don't use logo smaller than minimum sizes
❌ Don't add effects like shadows or outlines (unless specified)
