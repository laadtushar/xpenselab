# XpenseLab Brand Guidelines

## Overview

XpenseLab is a modern financial management application. This guide ensures consistent brand representation across all touchpoints.

## Brand Identity

### Logo Icon
The logo icon represents data flow and financial tracking through two overlapping bars:
- **Visual**: Two rounded bars intersecting at angles
- **Meaning**: Data streams, financial tracking, connectivity
- **Colors**: Gradient from electric blue through primary blue to emerald green

### Brand Name
- **Format**: Always lowercase `xpenselab`
- **Typography**: Manrope Bold
- **Spacing**: Tight letter spacing (-0.033em)

## Logo Usage

### Primary Logo (Stacked)
**Best for**: 
- Main brand identity
- Dark backgrounds
- Hero sections
- Marketing materials

**Specifications**:
- Icon above text
- Centered alignment
- Full color gradient

### Primary Logo (Light Background)
**Best for**:
- Light backgrounds
- Print materials
- White/light surfaces

**Specifications**:
- Top accent bar (gradient)
- Icon and text in full color
- Dark text on light background

### Monochrome Logo
**Best for**:
- Single-color applications
- Fax, embossing, engraving
- Low-color contexts

**Specifications**:
- White version for dark backgrounds
- Dark version for light backgrounds
- No gradients

### Horizontal Lockup
**Best for**:
- Navigation bars
- Headers
- Wide horizontal spaces
- Mobile apps

**Specifications**:
- Icon and text side-by-side
- Divider line between icon and text
- Compact horizontal format

### App Icon
**Best for**:
- Mobile app icons
- Favicons
- Social media avatars
- App store listings

**Specifications**:
- Square format (1:1)
- Rounded corners (14% radius)
- Icon only (no text)
- Full color gradient

## Color Usage

### Primary Colors
Use primary blue (`#0d59f2`) for:
- Primary CTAs
- Active states
- Links
- Brand accents

### Gradient Usage
Use gradients for:
- Logo icon
- Accent elements
- Progress indicators
- Special highlights

### Background Colors
- **Light Mode**: Use light background (`#f5f6f8`) with dark text
- **Dark Mode**: Use dark background (`#101622`) with light text

## Spacing & Sizing

### Minimum Sizes
- **Primary Logo**: 32px height minimum
- **Horizontal Logo**: 120px width minimum
- **App Icon**: 16x16px (favicon), 512x512px (app store)

### Clear Space
Maintain clear space equal to the icon height around all sides of the logo.

### Scaling
Always maintain aspect ratio when scaling logos.

## Do's and Don'ts

### ✅ Do's
- Use full-color logos on appropriate backgrounds
- Maintain clear space around logos
- Use correct logo variation for context
- Follow minimum size requirements
- Use lowercase for brand name

### ❌ Don'ts
- Don't rotate or distort logos
- Don't change logo colors (except monochrome)
- Don't place logos on busy backgrounds
- Don't use logos smaller than minimum sizes
- Don't add effects (shadows, outlines) unless specified
- Don't capitalize brand name
- Don't use incorrect logo variations

## File Formats

### SVG
- **Use**: Web, scalable graphics
- **Advantage**: Vector, scalable, small file size

### PNG
- **Use**: Print, raster graphics, transparency needed
- **Sizes**: Multiple sizes available (16px to 512px+)

### ICO
- **Use**: Favicons only
- **Sizes**: 16x16, 32x32, 48x48
